# Poetry GPT Learning Site

This repository contains the public learning site for a Chinese classical
poetry GPT pretraining walkthrough.

## What is included

- Root static files: the GitHub Pages site.
- `tutorials/`: Markdown source tutorials.

## What is intentionally excluded

- Raw poetry datasets.
- Processed training corpora.
- Model checkpoints and model weights.
- Python virtual environments and local machine paths.

The published site is a static tutorial. It does not run model inference in the
browser.

## GitHub Pages setting

Use branch-based Pages publishing:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/root`
